Digital Signature


Welcome to Digital Signature

Our Website generates Digital Signature for a teacher and also verifies for the student that the digital Signature sent to be is sent by a rightful host.

Instruction for teacher:-
  • click on the generation button and sign up if you are a new user else login with your credentials.

  • you sign up for the first time you will receive a private key corresponding to your account which is meant to be kept confidential by the user.

  • Now the user will be directed to the page where they can create a Digital Signature corresponding to any file they want,

  • The user need to provide the file and private key, a Digital Signaturewill be created corresponding to that file which is unique

  • The user can now share their file to anyone along with the Digital Signature so that person can validate and check that if any discripense is there in the file provided with the original file
Instruction for student:-
  • user need not to sign up, they can directly go to the verification page by clicking on the verification button

  • There the user can provide with the Digital signature and the file you received and click the submit button

  • A message will be displayed indicating whether the file is manipulated by the third party or not.
